# Übungen – Einstieg in die objektorientierte Programmierung

## Übung 1 – Von Variablen zur Klasse

In einem Unternehmen werden die Daten eines Computers momentan in einzelnen Variablen gespeichert.

```java
String hersteller = "Dell";
String modell = "OptiPlex 7010";
int ram = 16;
boolean eingeschaltet = true;
```

### Aufgaben

1. Welche Informationen gehören logisch zusammen?
2. Warum könnten einzelne Variablen bei vielen Computern unübersichtlich werden?
3. Entwirf eine passende Klasse `Computer`.
4. Lege die benötigten Attribute fest.
5. Erstelle die Klasse in Java.

## Übung 2 – Objekte erzeugen

Erzeuge nun drei verschiedene Computer.

Verwende beispielsweise folgende Daten:

| Hersteller | Modell          |   RAM | Eingeschaltet |
| ---------- | --------------- | ----: | :-----------: |
| Dell       | OptiPlex 7010   | 16 GB |       Ja      |
| Lenovo     | ThinkCentre M75 | 32 GB |      Nein     |
| HP         | EliteDesk 800   |  8 GB |       Ja      |

### Aufgaben

1. Erzeuge drei Objekte.
2. Weise jedem Objekt passende Werte zu:
```java
Computer pc = new Computer();

pc1.hersteller = "Dell";
pc1.modell = "OptiPlex 7010";
pc1.ram = 16;
pc1.eingeschaltet = true;

```
3. Gib anschließend einzelne Attribute auf der Konsole aus.

Beispiel:

```text
Dell
16
true
```

## Übung 3 – Methoden

Bisher werden die Informationen einzeln ausgegeben.

Erweitere die Klasse `Computer` um eine Methode

```java
void gibInformationenAus()
```

Die Methode soll alle Informationen des Computers ausgeben.

### Beispielausgabe

```text
Hersteller: Dell
Modell: OptiPlex 7010
RAM: 16 GB
Eingeschaltet: Ja
```

Rufe die Methode anschließend für alle drei Computer auf.
