# Übungen – Einstieg in die objektorientierte Programmierung 2

## Übung 4 – Konstruktor

Momentan werden die Attribute nach dem Erzeugen eines Objekts gesetzt.

```java
Computer pc = new Computer();

pc.hersteller = "Dell";
pc.modell = "OptiPlex 7010";
pc.ram = 16;
pc.eingeschaltet = true;
```

### Aufgaben

1. Was ist an dieser Vorgehensweise umständlich?
2. Welche Fehler können passieren?
3. Erweitere die Klasse um einen passenden Konstruktor.
4. Erzeuge die drei Computer anschließend mithilfe des Konstruktors.

## Übung 5 - mehrere Konstruktoren

Wir wollen nun dafür sorgen, dass Computer in einer Standardkonfiguration erstellt werden oder aber individuell konfiguriert gebaut werden können.

In der vorigen Übung haben wir einen Konstruktor mit Parametern erstellt. Wir müssen jetzt aber auch bei der Erzeugung eines neuen Objekts dem Konstruktor immer konkrete Werte übergeben.

Erstellt einen weiteren Konstruktor, der nur einen Paramenter entgegennimmt, zusätzlich bestimmte Standardwerte setzt, z.B.:


| Hersteller | Modell          |   RAM | Eingeschaltet |
| ---------- | --------------- | ----: | :-----------: |
| bei der Erzeugung festzulegen       | Standard   | 8 GB |       Nein      |


Es soll also möglich sein, entweder ein Objekt mit diesen Standardwerten zu erzeugen, oder aber bei der Erstellung konkrete beliebige Werte anzugeben.

## Übung 6 – Mehrere Objekte verwalten

Die IT-Abteilung verwaltet nun viele Computer.

Lege ein Array für Computer an.

```java
Computer[] computer = new Computer[3];
```

### Aufgaben

1. Speichere deine drei Computer im Array.
2. Gib anschließend alle Computer mit einer Schleife aus.
3. Verwende dabei die Methode `gibInformationenAus()`.

## Übung 7 – Eigene Klasse entwerfen

Ein Unternehmen möchte zusätzlich seine Mitarbeitenden verwalten.

Überlege dir zunächst:

* Welche Informationen gehören zu einer Mitarbeiterin oder einem Mitarbeiter?
* Welche Attribute werden benötigt?

### Anforderungen

Die Klasse soll mindestens besitzen:

* vier Attribute
* mindestens zwei Methoden, z.B.: 
  - `gibInformationenAus()`
  - `urlaubBeantragen()`
  - `arbeitsplatzAusgeben()`
  - `abteilungWechseln(String neueAbteilung)`   
* einen Konstruktor

Erzeuge anschließend mindestens zwei verschiedene Mitarbeitende und gib ihre Informationen aus und rufe die implementierten Methoden auf.

## Zusatzaufgabe – IT-Inventar

Ein Büroraum soll vollständig beschrieben werden.

Folgende Geräte befinden sich darin:

* Computer
* Monitor
* Drucker

### Aufgaben

Entwirf **eine weitere Klasse** deiner Wahl.

Mögliche Klassen:

* `Monitor`
* `Drucker`
* `Laptop`
* `Beamer`
* `Switch`
* `Router`

#### Anforderungen

Die Klasse soll besitzen:

* mindestens vier Attribute
* einen Konstruktor für Standardwerte und einen weiteren Konstruktor
- mindestens zwei Methoden

Erzeuge anschließend mehrere Objekte dieser Klasse und rufe deren Methoden auf.
