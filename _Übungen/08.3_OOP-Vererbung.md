# Übung 8 – Unterklassen `Laptop` und `DesktopComputer`

## Aufgaben

1. Nutze deine vorhandene Klasse `Computer` als Oberklasse.

2. Erstelle eine Unterklasse `Laptop`, die von `Computer` erbt.

   Zusätzliche Attribute:

   - `akkuStand`
   - `bildschirmGroesse`

3. Erstelle eine weitere Unterklasse `DesktopComputer`, die von `Computer` erbt.

   Zusätzliche Attribute:

   - `hatRgbBeleuchtung`
   - `gehaeuseTyp`

4. Erstelle in beiden Unterklassen einen Konstruktor.

   Rufe im Konstruktor mit `super(...)` den Konstruktor der Oberklasse auf, damit Hersteller, Modell, RAM und der Einschaltzustand übernommen werden. Initialisiere anschließend die zusätzlichen Attribute der Unterklasse.

5. Überschreibe in beiden Unterklassen die Methode

   ```java
   gibInformationenAus()
   ```

    Die Methode soll auch die zusätzlichen Attribute der jeweiligen Unterklasse ausgeben.

6. Erzeuge jeweils zwei Objekte der Klassen Laptop und DesktopComputer.

    - Lege ein Array `geraete` vom Typ `Computer[]` an.
    - Speichere darin alle erzeugten Objekte:
        - beide Laptop-Objekte
        - beide DesktopComputer-Objekte

    Überlege dabei:

    > Warum dürfen Objekte der Klassen Laptop und DesktopComputer in einem Array vom Typ Computer[] gespeichert werden?

    Rufe für alle Geräte im Array die Methode `gibInformationenAus()` mit einer Schleife auf.
