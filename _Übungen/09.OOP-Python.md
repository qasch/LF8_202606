# Übung - Objektorientierung in Python

## Aufgabe 1 – Erste Klasse

Erstellt eine Klasse

```python
Computer
```

Die Klasse soll folgende Attribute besitzen:

* hersteller
* modell
* ram
* eingeschaltet

Verwendet einen Initializer (Konstruktor in Java) (`__init__()`), um die Attribute zu initialisieren.

## Aufgabe 2 – Objekte erzeugen

Erzeugt zwei Computer.

Beispielsweise:

* Dell OptiPlex
* Lenovo ThinkCentre

Gebt anschließend die Werte der Attribute auf der Konsole aus.

## Aufgabe 3 – Methoden

Erweitert die Klasse um folgende Methoden:

```python
einschalten()
```

```python
ausschalten()
```

```python
gib_informationen_aus()
```

Die Methode `gib_informationen_aus()` soll alle Informationen des Computers ausgeben.

## Aufgabe 4 – Vererbung

Erstellt eine Unterklasse

```python
Laptop
```

Die Klasse besitzt zusätzlich:

* akku_stand
* bildschirmgroesse

Verwendet den Initializer der Oberklasse mit

```python
super()
```

Initialisiert anschließend die zusätzlichen Attribute.

## Aufgabe 5 – Zweite Unterklasse

Erstellt eine weitere Unterklasse

```python
DesktopComputer
```

Zusätzliche Attribute:

* hat_rgb_beleuchtung
* gehaeuse_typ

Verwendet ebenfalls

```python
super()
```

im Initializer.

## Aufgabe 6 – Methoden überschreiben

Überschreibt in beiden Unterklassen die Methode

```python
gib_informationen_aus()
```

Die Ausgabe soll zusätzlich die Attribute der jeweiligen Unterklasse enthalten.

## Aufgabe 7 – Polymorphie

Erstellt eine Liste mit verschiedenen Geräten.

Beispiel:

```python
geraete = [
    ...
]
```

Die Liste soll enthalten:

* zwei Laptops
* zwei DesktopComputer

Gebt anschließend alle Geräte mit einer Schleife aus.

Verwendet dabei ausschließlich

```python
geraet.gib_informationen_aus()
```

Überlegt:

> Warum wird trotzdem die richtige Methode aufgerufen?

## Aufgabe 8 – Vergleich Java und Python

## Aufgabe 9 – Unterschiede erkennen

Notiert mindestens fünf Unterschiede zwischen Java und Python.

Denkt dabei beispielsweise an:

* Datentypen
