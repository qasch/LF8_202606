# Übungen – "Einfache" Subqueries

## Aufgabe 1 – Der größte Patient

Gib den Vor- und Nachnamen aller Patienten aus, die die **größte Körpergröße** besitzen.

### Hilfestellung

Überlege dir zunächst:

> Wie kannst du die größte Körpergröße ermitteln?

```sql
SELECT MAX(height)
FROM patients;
```

Anschließend musst du alle Patienten finden, deren `height` diesem Wert entspricht.

## Aufgabe 2 – Größer als der Durchschnitt

Gib alle Patienten aus, deren Körpergröße **über dem Durchschnitt** aller Patienten liegt.

Ausgabe:

- Vorname
- Nachname
- Körpergröße

### Hilfestellung

Die innere Abfrage soll den Durchschnitt berechnen.

```sql
SELECT AVG(height)
FROM patients;
```

Diesen Wert kannst du anschließend im `WHERE` verwenden.

## Aufgabe 6 – Patienten mit einer Aufnahme

Gib alle Patienten aus, die mindestens einmal aufgenommen wurden.

Ausgabe:

- Vorname
- Nachname

### Hilfestellung

Schau dir zunächst an, welche Patientennummern in der Tabelle `admissions` vorkommen.

```sql
SELECT patient_id
FROM admissions;
```

Diese Liste kannst du anschließend verwenden.

## Aufgabe 8 – Ärzte mit mindestens einer Aufnahme

Gib alle Ärzte aus, die bereits mindestens einen Patienten behandelt haben.

Ausgabe:

- Vorname
- Nachname

### Hilfestellung

In der Tabelle `admissions` findest du die Spalte

```text

```

Diese enthält die `doctor_id` des behandelnden Arztes.

## Aufgabe 10 – Patienten aus derselben Provinz

Patient mit der `patient_id = 10` wohnt in einer bestimmten Provinz.

Gib alle Patienten aus, die in derselben Provinz wohnen.

Ausgabe:

- Vorname
- Nachname
- province_id

### Hilfestellung

Ermittle zunächst die `province_id` des Patienten.

```sql
SELECT province_id
FROM patients
WHERE patient_id = 10;
```

Verwende dieses Ergebnis anschließend in einer zweiten Abfrage.

## Aufgabe 11 – Längster Krankenhausaufenthalt

Gib alle Aufnahmen aus, deren Aufenthaltsdauer am längsten war.

Ausgabe:

- patient_id
- admission_date
- discharge_date

### Hilfestellung

Die Aufenthaltsdauer kannst du in SQLite berechnen mit

```sql
julianday(discharge_date) - julianday(admission_date)
```

Berechne zunächst die maximale Aufenthaltsdauer.

## Aufgabe 12 – Aufnahme länger als der Durchschnitt

Gib alle Krankenhausaufenthalte aus, deren Dauer über dem Durchschnitt liegt.

Ausgabe:

- patient_id
- admission_date
- discharge_date

### Hilfestellung

Berechne zunächst den Durchschnitt der Aufenthaltsdauer.

Die Berechnung ist dieselbe wie in Aufgabe 11.

*Tipp:* Du benötigst zwei Subqueries.

## Eine Subquery kann ...

### ... einen einzelnen Wert liefern

```sql
WHERE height >
(
    SELECT AVG(height)
    FROM patients
)
```

### ... eine Liste liefern

```sql
WHERE patient_id IN
(
    SELECT patient_id
    FROM admissions
)
```

> Eine Subquery ersetzt einen Wert oder eine Liste, die sonst von Hand eingetragen werden müsste.
